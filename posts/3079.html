Vue 数据操作 — Mustache

如何将 data 中的文本数据插入到 HTML 中呢 ？
可以通过 Mustache 语法（也就是双大括号）。

Mustache 基本使用

我们可以像下面这样使用，并且数据是响应的：

<body>
  <div id="app">
    <!-- 插入标签中 -->
    <div>hello {{name}}</div>
    <!-- 使用两个Mustache -->
    <div>{{firstName}} {{lastName}}</div>
    <!-- 也可以是一个表达式 -->
    <div>{{firstName + ' ' + lastName}}</div>
    <div>{{counter * 2}}</div>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    var vm = new Vue({
      el: "#app",
      data: {
        name: "VueJs",
        firstName: "zhang",
        lastName: "san",
        counter: 100,
      },
    });
  </script>
</body>

通过上述基本使用样例，可以看到，在 Mustache 语法中，可以有以下用法：

  • 直接在标签中插入。
  • 使用两个 Mustache 进行拼接。
  • 在 Mustache 中使用表达式。

v-once 指令

当某些情况下，可能不希望界面随意的跟随数据改变，就可以使用 v-once 指令。

  • 该指令后面不需要跟随任何表达式（比如之前的 v-for 后面是有表达式的）。
  • 该指令表示元素和组件只渲染一次，不会随着数据的改变而改变。

代码样例：

<body>
  <div id="app">
    <div>{{msg}}</div>
    <div v-once>{{msg}}</div>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    var vm = new Vue({
      el: "#app",
      data: {
        msg: "Vuejs",
      },
    });
  </script>
</body>

v-html 指令

当某些情况下，从服务器请求到的数据本身就是一个 HTML 代码。

如果直接通过 Mustache 来输出，会将 HTML 代码一并输出，但是我们希望的是按照 HTML 格式进行解析，并且显示对应的内容。

如果我们希望解析出 HTML 展示，就可以使用 v-html 指令，该指令后面旺旺跟上一个 string 类型，vue 实例对象会将 string 的 html 解析出来并进行数据渲染。

代码样例：

<body>
  <div id="app">
    <div>{{url}}</div>
    <div v-html="url"></div>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    var vm = new Vue({
      el: "#app",
      data: {
        url: '<a href="https://www.baidu.com/">百度一下</a>',
      },
    });
  </script>
</body>

v-text 指令

v-text 作用和 Mustache 比较相似：都是用于将数据显示在界面中。

通常情况下，接受一个 string 类型。

代码样例：

<body>
  <div id="app">
    <div>{{msg}}</div>
    <div v-text="msg"></div>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    var vm = new Vue({
      el: "#app",
      data: {
        msg: "hello world",
      },
    });
  </script>
</body>

执行结果：

v-text 和 Mustache （双大括号）结果一样，需要注意的是，当使用 Mustache 时，可以进行数据的拼接等操作，而 v-text 则不可以。

v-pre 指令

当不需要解析 Mustache 语法时，可以使用 v-pre 来跳过这个元素和它子元素的编译过程，用于显示原本的 Mustache 语法。

比如下面的代码：

第一个 div 元素中的内容会被编译解析出来对应的内容。

第二个 div 元素中会直接显示 。

代码样例：

<body>
  <div id="app">
    <div>{{msg}}</div>
    <div v-pre>{{msg}}</div>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    var vm = new Vue({
      el: "#app",
      data: {
        msg: "hello world",
      },
    });
  </script>
</body>

v-cloak 指令

由于浏览器是先加载 HTML 结构，再渲染数据，所以，当频繁刷新页面时，可以看到标签的闪动，此问题可通过 v-cloak 指令进行解决。

这个指令保持在元素上直到关联实例结束编译。和 CSS 规则如 [v-cloak] { display: none } 一起用时，这个指令可以隐藏未编译的 Mustache 标签直到实例准备完毕。

代码样例：

  • CSS 代码
<style>
  [v-cloak] {
    display: none;
  }
</style>
  • HTML 代码
<body>
  <div id="app">
    <div v-cloak>{{msg}}</div>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    var vm = new Vue({
      el: "#app",
      data: {
        msg: "hello world",
      },
    });
  </script>
</body>
